DIYPS

The objective of this project was to create our own 'photoshop' using the p5.js library, and create a set of brushes that either focus on aesthetics, culture or politics.

Go ahead and draw anywhere on the page.

1 - thin line
2 - thick line
3 - small triangle
4 - large triangle
5 - small square
6 - large square
7 - small ellipse
8 - large ellipse
9 - grey square ( opaque for creating different shades of grey)
0 - colorful circles (if it's going to be used, use it first. mostly for creating a colorful backdrop)
v - value text
